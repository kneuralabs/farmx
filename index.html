Getting Started

How to Use This Preview

Switching Roles

Use the pill switcher at the top to move between dashboards. This is a connected prototype: actions in one role update the others in real time — try requesting a booking as Enterprise, then check the Vendor and Admin tabs.

Enterprise Dashboard

Check your stats

Active relationships, pending bookings, and expiring documents update live as records change.

Manage the relationship ledger

Vendors are grouped by stage — Preferred, Active, Pending, Archived — your permanent partnership history.

Request bookings & upload documents

Submissions here immediately appear in the Vendor and Admin views.

Vendor Directory

Search or filter

Use the search bar to narrow the vendor list by name or category.

Update your profile

Changes here reflect instantly in the Enterprise ledger and vendor grid.

Respond to requests

Accepting or declining updates the booking status back on the Enterprise dashboard.

Admin Panel

Monitor activity

The activity feed logs every booking, document, and profile change platform-wide.

Review applications

Approving or rejecting an enterprise updates its status everywhere it's shown.

Notes

Every form is labeled with an Input: Role tag. Records with ✎ / ✕ can be edited or deleted inline — changes propagate across all views since every screen reads from the same shared data.
